# 3D Tilt Food Cards 🍽️

An interactive web application featuring stunning 3D tilting food cards that showcase delicious dishes with smooth animations and hover effects! 🍝🥗🍕

## Features ✨
- **3D Tilt Effect**: Cards tilt dynamically on mouse hover for an immersive experience
- **Responsive Design**: Optimized for all devices and screen sizes
- **Smooth Animations**: CSS-powered transitions for fluid interactions
- **Food Showcase**: Featuring Pasta Primavera, Caesar Salad, and Margherita Pizza

## Live Demo 🌐
Check out the live demo: [https://iam269.github.io/3D-Tilt-Food-Cards/](https://iam269.github.io/3D-Tilt-Food-Cards/)

## Technologies Used 🛠️
- HTML5
- CSS3 (with custom animations)
- JavaScript
- Google Fonts (Poppins)
